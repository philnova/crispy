About This Tool

crispy is a simple, fast, and open-source alternative to existing GuideRNA design tools for the CRISPR/CAS9 gene editing system.

Simple

crispy provides three ways to design GuideRNAs. As with other GuideRNA design tools, you can simply paste your gene sequence into the Raw Sequence Search tab. In addition, you can also search by UCSC genomic coordinates and even gene names.

Fast

crispy is designed to find GuideRNAs fast so you can get back to the bench. We do this by pre-computing and scoring all of the candidate guide sequences in the genome, so if you search by gene name or coordinate, finding the best guides is as easy as scanning through our database.

We recognize that you may need to search against custom nucleotide sequences, so we also provide the option of searching against a raw sequence. Since we have to do the off-target analysis from scratch, this method doesn't work as fast (but you'll still get the information you need).

Open Source

We believe that science should be shared freely. That's why all the code for crispy is hosted publicly on github so you can see how the site operates. crispy is built using Flask, the microframework for Python web development, and we use SQLAlchemy for database management. There's also some jQuery to make the site look nice.

If you believe we could improve our algorithms or would like to request additional features, reach out to us (see contact info below) and we'll do our best.

Algorithm

Our scoring algorithm is drawn from Zhang et al. 2013, which accounts both for the number of off-target matches a putative guide sequence has in the genome, the pairwise distance between those mismatches, and the position of the mismatches within the sequence.
